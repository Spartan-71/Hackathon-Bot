# Privacy Policy

**Last Updated: January 15, 2026**

## 1. Introduction

This Privacy Policy explains how HackRadar ("we", "our", "the Bot") collects, uses, and protects your information when you use our Discord bot.

## 2. Information We Collect

### 2.1 Automatically Collected Information

When you use the Bot, we automatically collect:

- **Server Information:**
    - Server ID (Discord's unique identifier for your server)
    - Channel IDs where the Bot is active
    - Server name (for display purposes only)
- **User Information:**
    - User ID (Discord's unique identifier)
    - Username (for interaction purposes)

### 2.2 Information You Provide

When you use specific features, we may collect:

- **Notification Settings:**
    - Designated notification channels (via `/set_channel` command)
    - Server preferences
- **Search Queries:**
    - Keywords you search for (via `/search` command)
    - Platform filters you use (via `/platform` command)
- **Subscriptions:**
    - Themes you subscribe to (via `/subscribe` command)

### 2.3 Information We Do NOT Collect

We do NOT collect:

- Message content (except direct commands to the Bot)
- Personal conversations
- Files or attachments
- Email addresses or phone numbers
- Financial information
- Real names or addresses

## 3. How We Use Your Information

We use collected information to:

- **Provide Bot Functionality:**
    - Send hackathon notifications to designated channels
    - Process your commands and queries (`/fetch`, `/search`, `/platform`)
    - Deliver hackathon information in response to commands
- **Improve the Service:**
    - Analyze usage patterns to improve features
    - Fix bugs and technical issues
    - Develop new features

## 4. Data Storage and Security

### 4.1 Storage

Your data is stored:

- **Database:** PostgreSQL database
- **Hosting:** Railway
- **Duration:** Data is retained as long as the Bot is in your server or until you request deletion
- **Backup:** Regular backups are made for disaster recovery

### 4.2 Security Measures

We implement security measures including:

- Encrypted database connections
- Secure server access controls
- Regular security updates
- Limited data access (only authorized developers)

However, no method of transmission over the internet is 100% secure.

## 5. Data Sharing

### 5.1 We Do NOT Sell Your Data

We will NEVER sell, rent, or trade your information to third parties.

### 5.2 Third-Party Services

We use the following third-party services:

- **Hosting Provider:** Railway
    - Purpose: Host the Bot and database
    - Data shared: Technical server data only
- **Hackathon Platforms:** Unstop, Devpost, Devfolio, Dorahacks and MLH
    - Purpose: Fetch public hackathon information
    - Data shared: None (we only read publicly available data)

### 5.3 Legal Requirements

We may disclose information if required by law or to:

- Comply with legal processes
- Protect our rights or property
- Prevent fraud or abuse

## 6. Your Rights and Choices

### 6.1 Data Access

You can request to see what data we have about you by contacting us.

### 6.2 Data Deletion

You can request deletion of your data by:

- Removing the Bot from your server (automatically deletes server-specific data)
- Contacting us directly for specific data deletion requests

When you remove HackRadar from your server, we delete:

- Server ID
- Channel configurations
- All associated server data within 30 days

### 6.3 Opt-Out

You can opt-out of data collection by:

- Not using the Bot
- Removing the Bot from your server

## 7. Children's Privacy

The Bot is not intended for children under 13. We do not knowingly collect data from children under 13. If you believe a child under 13 has provided data, please contact us immediately.

## 8. International Data Transfers

Your data may be stored on servers located in US]. By using the Bot, you consent to this transfer and storage.

## 9. Data Retention

We retain your data:

- **Active Servers:** As long as the Bot is in your server
- **Removed Servers:** Data is deleted within 30 days of Bot removal
- **Usage Logs:** Kept for up to 90 days for debugging and analytics
- **Error Logs:** Kept for up to 30 days

## 10. Cookies and Tracking

The Bot does NOT use cookies or tracking technologies. All data collection happens through Discord's API during your direct interactions with the Bot.

## 11. Bot Commands and Data Collection

Here's what data each command collects:

- **`/hi`** - No data stored
- **`/fetch`** - Command usage logged (no personal data)
- **`/search [keyword]`** - Search query logged for analytics
- **`/platform [name]`** - Platform preference logged
- **`/set_channel`** - Channel ID and Server ID stored for notifications
- **`/subscribe [theme]`** - User ID and theme preference stored
- **`/unsubscribe [theme]`** - Subscription data removed
- **`/upcoming [days]`** - No data stored
- **`/help`** - No data stored
